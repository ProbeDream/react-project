### TodoList 精美任务列表
#### 1. 使用方式
- 直接通过Git clone 克隆项目到本地
- 通过 Yarn install 或者说 npm install 下载对应的依赖
- 最后通过 Yarn start 或者说 npm run start 启动项目
- 最后通过分配的端口 进入到对应的页面!

#### 2. 使用到的技术栈

React + LeanCloud + CSS 

通过React做UI框架,对应的后端的数据获取是通过对接LeanCloud的API接口对数据进行操作!
